# Base de Datos - Lista de Contactos Personales

## Descripción
Este proyecto contiene el diseño y la implementación de una base de datos de contactos personales.
Incluye el modelo entidad/relación, la creación de tablas, relaciones y datos de prueba.

## Entidades
- persona: información principal del contacto
- direccion: direcciones asociadas a cada persona
- categoria: tipo de relación (Familia, Amigos, Trabajo)
- contacto_categoria: relación entre persona y categoría
